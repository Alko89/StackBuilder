StackBuilder

StackBuilder is a simple palettization software (homogeneous case stacking).

It helps you to solve some problems like :

  • palettization : define a case and a pallet then find the best layout to stack the case,
  • case filling : define a primary packaging and a case or a set of cases then find the best case and the best way to fill it,
  • truck load : select a paletizing solution then optimize the truck/container load,
  • optimal case search : given a box, a number of box per case and a pallet type, find the optimal case layout and dimensions.
