FAQ

What kind of problems can StackBuilder help me to solve?
StackBuilder is aimed at solving homogeneous stacking problems, which means it can ONLY solve problems where a single item (box/case) is involved.
Also, it builds solutions using homogeneous layers in which the vertical axis remains the same. It can not provide solutions in which a "turned" case is used to fill remaining space.

How can I change unit system?
Go to menu "Tools\Settings"
In the Options dialog that opens, select "Options\Unit System".
Select the system you would like to use in the "Unit System" combo.
Click button OK and close StackBuilder. Restart StackBuilder so that change is taken into account.
How can I generate a report?
Once you have an analysis, select a solution by either checking it in the "Selected" column of the solutions array or pressing the "Select" button on the analysis form.
A small light bulb icon should appear under the analysis node of the StackBuilder tree.
Right-click the light bulb icon,
In the context menu that appears, select "Generate Word Report for..."
Microsoft Word should open with your report loaded.
