How to update et generate Help file ?
-------------------------------------

- Download HelpNDoc (http://www.helpndoc.com/download)
- Open Stackbuilder.hnd
- Generate TreeDim.StackBuilder.Desktop.chm
- Copy in directory .\Decompile
- Decompile using Microsoft Html Help Workshop
- Remove any advertising
- Recompile using Microsoft Html Help Workshop
